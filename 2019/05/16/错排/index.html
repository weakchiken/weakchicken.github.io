###错排：

公式：a[i]=(i-1)*(a[i-1]+a[i-2])

链接：https://blog.csdn.net/achesong/article/details/87387024

错排公式的原形为D(n) = n! (1/0! - 1/1! + 1/2! - 1/3! - ….. + (-1)^n/n!)，当n很大时计算就很不方便。一个供参考的简化后的公式是D(n) = [n!/e+0.5] ，其中e是自然对数的底，[x]为x的整数部分。
